# KEEPER.

A productivity-focused tab manager that remembers **why** you opened a tab.

Most people open too many tabs and eventually forget their purpose.  
Keeper solves that problem by letting users save:
- the link
- the reason/context
- reminders
- deadlines
- productivity tracking

---

# Features

- Save tabs with context
- Category filters
- Search saved tabs
- Real-time countdown timers
- Browser notifications
- Deadline mode
- Productivity analytics
- Pin important tabs
- Mark tasks as done
- Local storage persistence
- Open saved links instantly
- Auto website favicons

---

# Tech Stack

- React
- JavaScript
- CSS
- Vite
